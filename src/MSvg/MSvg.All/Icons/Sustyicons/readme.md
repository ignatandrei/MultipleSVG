From https://sustyicons.com/ (source repository: https://github.com/codewordcreative/susty-icons).

Use `download-sustyicons.ps1` to download:

- all SVG files from `source-icons/`
- the upstream `LICENSE` file

The script also verifies:

- SVG file count matches the source tree count
- downloaded license contains `Copyright Notice` and `Permission is hereby granted`

For local/offline validation, optional parameters `TreeFilePath`, `SourceRootPath`, and `LicenseSourcePath` can be used.
